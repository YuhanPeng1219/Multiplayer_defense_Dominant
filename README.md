# Multiplayer_defense_Dominant
 Using dominant region to solve multiplayer defense game

1. *runsim_point.m* and *runsim_rect.m* are used to generate all the data needed for a video. The target and initial position of players can be changed in these two scripts. Run the two first.

2. *Sim_video.m* is for animation.

3. Remember to copy the *data_N100_mesh10000.mat* into directory data before running simulation.
